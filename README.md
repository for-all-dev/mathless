## Stack

- Next.js App Router with MDX for prose (`content.mdx` is rendered inside `app/page.tsx`).
- Tailwind CSS via `@tailwindcss/postcss`.
- Geist / Geist Mono via `next/font`.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Edit `content.mdx` for prose, `app/page.tsx` for the hero, and `app/layout.tsx` for metadata.
